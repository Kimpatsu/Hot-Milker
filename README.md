# Hot-Milker
A script made to generate Hot Milk meme from two images.

# How to use:

1. Import it to your python script. Call the only function ```hotmilkify()```
2. Profit

Example
```python
#imports hotmilker
import hotmilker

#Asks the susers the images
clip1 = input("Image 1: ")
clip2 = input("Image 2: ")

#Profit
hotmilker.hotmilkify(clip1, clip2)
```
